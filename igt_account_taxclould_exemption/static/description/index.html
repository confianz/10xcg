Tax Cloud Exemption Certificate Integration


Enter exemption certificate purchase details and Get Exemption Certificate Id from tax Cloud.
Use the certificate on sales order to get exemption and pass the same details on Invoice recompute and get exemption.



Features:
  • Enter Exemption Certificate details and Get Exemption Certificate ID from Tax Cloud
  • Get tax exemption on sales order by selecting certificate ID
  • Get tax exemption on Invoice by selecting certificate ID
Steps how you can use this module:
  • Goto Accounting -> Accounting -> Miscellaneous -> Tax Exemption Certificates.
  • Enter Tax Exemption Certificate details and get exemption ID.
  • Enter purchaser details to get exemption ID.
  • Create a Sales order, here you can select the tax exemption ID related the the selected customer. If you select the exemption id, it will check in tax cloud and apply the exemption.
  • Yan get the exemption on the Invoice as well. To get the exemption one must select the Tax Exemption id and when you hit confirm system will pass the exemption id on tax cloud and check for exemption and apply on related invoice. If your invoice is created from sales order than it will pass tax exemption certificate from Sales order to invoice, but before confirming the invoice you can change it.
